WebsiteReliability使用文档

本项目基于Vue Elemenet-Ui 和PHP实现的可以托管在虚拟主机上的一款URL检测平台
本项目仍处于开发状态，需要大量的测试来修改评判标准，现在已经发布了最简易的基础版本，后续会加上管理员后台等等。
